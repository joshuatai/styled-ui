Badge

Badges are used to highlight an item's status for quick recognition.

Import

import Badge from '@trendmicro/react-styled-core/Badge';
// or
import { Badge } from '@trendmicro/react-styled-core';

Usage

100
<Badge>100</Badge>
EDITABLE EXAMPLE

Badge Color

  • Pass the variantColor prop to customize the color of the Badge. variantColor can be any color key with key 60 that exists in the theme.colors.

    Thats mean the green.50 exists in the theme.colors then can use variantColor="green".

  • Learn more about theming.

Props

Badge composes the Box component. You can override the default styles with style props.

NameTypeDefaultDescription
variantColorstring'gray'The color scheme to use for the badge. It must be a color key defined in theme.colors.
