ButtonGroup

ButtonGroup is used to group related buttons.

Import

import ButtonGroup from '@trendmicro/react-styled-core/ButtonGroup';
// or
import { ButtonGroup } from '@trendmicro/react-styled-core';

Usage

Basic button group

<ButtonGroup>
  <Button>One</Button>
  <Button>Two</Button>
  <Button>Three</Button>
</ButtonGroup>
EDITABLE EXAMPLE

Group variants

Use the variant prop to change the visual style of every button in a group. You can set the value to solid, outline. Use outline to remove all background images and colors on any button.

<Stack direction="column" spacing="1rem">
  <ButtonGroup variant="solid">
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
  </ButtonGroup>
  <ButtonGroup variant="outline">
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
  </ButtonGroup>
</Stack>
EDITABLE EXAMPLE

Group colors

Use the variantColor prop to change the color scheme of every button in a group. variantColor can be any color key with key 50(hover), 60(normal) and 70(active) that exist in the theme.color.

<ButtonGroup variantColor="blue">
  <Button>Left</Button>
  <Button>Middle</Button>
  <Button>Right</Button>
</ButtonGroup>
EDITABLE EXAMPLE

Group sizes

Use the size prop to change the size of every button in a group. You can set the value to sm, md, or lg.

<Stack direction="column" spacing="1rem">
  <ButtonGroup size="sm">
    <Button>Left</Button>
    <Button>Middle</Button>
    <Button>Right</Button>
  </ButtonGroup>
  <ButtonGroup size="md">
    <Button>Left</Button>
    <Button>Middle</Button>
    <Button>Right</Button>
  </ButtonGroup>
  <ButtonGroup size="lg">
    <Button>Left</Button>
    <Button>Middle</Button>
    <Button>Right</Button>
  </ButtonGroup>
</Stack>
EDITABLE EXAMPLE

Group orientation

Make a set of buttons appear vertically stacked rather than horizontally, by adding vertical to the ButtonGroup component.

<Stack direction="column" spacing="1rem" shouldWrapChildren>
  <ButtonGroup>
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
  </ButtonGroup>
  <ButtonGroup vertical>
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
  </ButtonGroup>
</Stack>

Props

ButtonGroup composes the Box component. You can override the default styles with style props.

NameTypeDefaultDescription
sizesm, md, lgmdSets the size for all Buttons in the group.
variantsolid, outlinesolidSets the variant of all Buttons in the group.
variantColorstringSets the variant of all Buttons in the group. Use a color key passed in theme.colors.
verticalbooleanfalseMake the set of Buttons appear vertically stacked.
